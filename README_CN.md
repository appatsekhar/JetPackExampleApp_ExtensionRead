# 拓意阅读JetPack版

一款清新简单的阅读类App，应用数据来源开眼api。
项目中采用了QMUI + Jetpack MVVM的架构。使用QMUI也是一种尝试，实际用过之后感觉该UI库不符合预期，并不推荐大家在商业化项目中使用。

[English Click](https://github.com/toeii/JetPackExampleApp_ExtensionRead/blob/master/README.md)
